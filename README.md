# eCommerce Inventory API

## Instructions

`npm install`  
`npm fund`  
`npm start`

Browse to: `http://localhost:3000/api/products/`

## Routes

**GET** `/api/products` - List all products  
**POST** `/api/products/` - Add a product  
`http://localhost/p2/4-17/products.php` - See inventory  
`http://localhost/p2/4-17/inventory_report` - Show stock levels less than 5  
`http://localhost/p2/4-17/consume_api.php` - See products formatted in _HTML/CSS_
